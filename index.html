// MANIFESTO

Os dados de aeronaves civis são públicos por natureza — transponders broadcast em aberto, voluntários captam, comunidades compartilham. Acreditamos que o acesso a esses dados deve permanecer público, gratuito e auditável. A API fica aberta; o código fica com o estúdio que opera. Abertura é dos dados, não da implementação.
