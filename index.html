CADA PROVEDOR
FALA UM IDIOMA
DIFERENTE.

Você quer dados de aeronaves no mundo todo. Os melhores feeds são públicos — mas cada um chega em um formato próprio, com cobertura regional diferente e regras de acesso próprias. Normalizar isso é o nosso trabalho.

ADSBExchange
Comunidade global · ~10k feeders · MLAT
format  binCraft + zstd
size    112 bytes / aeronave
lang    binário
delta   ~1s
PROVEDOR — B
OpenSky
Acadêmico · ETH Zurich · OAuth2
format  states/all (JSON)
shape   [hex,call,lon,lat,alt…]

// MANIFESTO

Os dados de aeronaves civis são públicos por natureza — transponders broadcast em aberto, voluntários captam, comunidades compartilham. Acreditamos que o acesso a esses dados deve permanecer público, gratuito e auditável. Por isso o código é aberto, a API é grátis, e nada disso depende de uma empresa.
